Estimado alumno:
Bienvenido al curso Manejo Estrategias de Manejo desde Atención Primaria del paciente espástico.Para superarlo y obtener el certificado es necesario aprobar el examen final, al que sólo se podrá acceder si se realiza todo el itinerario del curso. Es decir, no se podrá pasar al Módulo II sin haber realizado previamente el Módulo I, y así sucesivamente. Al finalizar cada módulo se podrán realizar los ejercicios de autoevaluación, que permitirán aplicar el contenido y preparar el examen final. Estos ejercicios no puntúan para la nota final. En el apartado "Material" está disponible la documentación de cada módulo.

El examen final estará disponible
hasta el 13 de diciembre.

Las dudas sobre el contenido del curso se resolverán en el apartado Preguntas.
Este curso tiene solicitada acreditación a la Comisión de Formación Continuada de las profesiones sanitarias.
El certificado del curso estará disponible una vez se resuelva la acreditación.
